This project is an emulator for a 8-bit CPU, specifically the 6502 processor. I have been using this site for reference: 

https://web.archive.org/web/20210501043555/http://www.obelisk.me.uk/6502/reference.html#LDA

This project is to increase my understanding of how a CPU works and I figured a modern CPU would be too difficult and time consuming to code.
